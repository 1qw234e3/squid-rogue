# 03 设计议题与创想

> 基于 M0 手感切片的实际体验与 v0.2 设计文档的交叉审视。
> 状态标注:✅ 已实现 / 📋 已排期 / 💡 待讨论

## 1. M0 暴露的设计问题

### 1.1 潜行是"二值"的,没有恢复回路 ✅
守卫一旦发现就永久追捕,失误一次潜行局就被迫转战斗。潜行张力来自
"被发现 → 逃脱 → 重新隐蔽"的循环,缺后半段。

**修法(已实现)**:守卫状态机扩为 巡逻 → 追捕 → 搜索 三态。
- 追捕只追"最后目击点",不再全图透视玩家位置;
- 跟丢 5 秒、或赶到最后目击点仍没人 → 转入搜索(视锥变橙色),原地张望 8 秒;
- 搜索期间视锥扫到玩家 → 重新追捕;搜不到 → 回巡逻。

### 1.2 守卫开枪零前摇,近距离不公平 ✅
被打中时玩家觉得"没法反应",是挫败不是难度。

**修法(已实现)**:开枪前 0.3 秒亮红色预瞄线,方向在前摇开始时锁定——
移动/翻滚可以躲。与木头人关"激光预瞄 0.5s"是同一设计语言,提前练习读条。

### 1.3 反杀一旦成立,潜行就作废 ✅
捡到强武器后见人就杀永远最优,杀守卫除了叫人没有别的代价。

**修法(已实现)**:噪音系统。开枪产生噪音圈(白色扩散环可视化),
圈内守卫即使没看见你也会赶来查看声源(进入搜索态)。
"杀这个守卫值不值"从此每次都是真决策;弩(无声)、消音鞋立刻有了存在意义。

噪音半径初值(进 §12 调参):手枪 130 / 冲锋枪 110 / 霰弹 180 / 守卫步枪不触发。

**这套系统是全游戏级的**:沙盒偷窃失手的动静、通风管爬快了的声响、
木头人关掩体被打碎的巨响,未来全走同一接口(EventBus.noise_emitted)。

### 1.4 BSP 迷宫没有地标,找出口靠瞎逛 ✅
房间全长一个样,迷路的烦躁会污染手感判断。

**修法(已实现)**:随机三个房间地面染色(紫/绿/红)作地标;
玩家身边常驻一枚半透明绿色箭头指向出口(出口进屏即隐藏)。

## 2. 设计文档(v0.2)层面的结构性风险

### 2.1 死在 R4 的代价不对称 💡 → 解法见创想 3
25 分钟社交投资,弹珠局被算计死,个人账户 300 块抚慰不了。灵魂观战太被动。

### 2.2 好感度数值化会被"刷" 💡
玩家会发现"送 5 次食物 = 生死之交",社交变成喂狗粮。建议:
- 好感只露**模糊段位**(对话语气、肢体动作),精确数值永不显示;
- 每个 NPC 一两个隐藏偏好(给酒鬼送酒 > 送食物),"读人"本身成为玩法。
比数值条更贴"人比关卡危险"支柱。

### 2.3 猜单双的像素脸特写是内容黑洞预备队 💡
8 性格 × 说谎模式 × 特写帧动画,量产成本与对话模板同量级。
MVP 用"心跳像素条"(已在宝物列表)做信息泄露通道,脸特写降级为 1~2 个通用帧。

## 3. 创想(按"能长在现有机制上"排序)

### 3.1 噪音地图——潜行统一货币 ✅(M0 版已落地,见 1.3)

### 3.2 广播插播"规则变更" 📋 M1
广播从演出升级为机制:小游戏进行中**实时修改规则**——
木头人打到一半广播"本轮新增:禁止蹲伏";抢椅子音乐第二段宣布"椅子数减半"。
变体随机从"开局抽一个"升级为"过程中还会抽"。
实现成本低:每关的变体接口多挂一个触发时机。

### 3.3 死后的"遗产时刻" 📋 M3
死亡瞬间 10 秒选择:把全部配给券 + 一件道具**遗赠**给场上任一好感 ≥50 的 NPC。
- 受赠 NPC 获得增益,图鉴记忆多一条"继承了 X 的遗物";
- 它若最终夺冠,你拿冠军奖金 15%。
死亡从"全损"变成"换一种方式下注",观战从看戏变成看自己的遗产能走多远。
直接强化"死了也有收获"支柱,顺便解 2.1。

### 3.4 守卫制服潜入线 📋 M5 后
"伪装面具 15s"太一次性。升级:沙盒放倒守卫可扒制服,穿上后以守卫身份
自由行动——但必须遵守排班表(偷来看的那张表反过来约束你),
走错巡逻路线、被同班搭话答不上暗号即穿帮。
排班表从"情报道具"升级成"双向玩法"。成本主要是几条暗号对话模板。

### 3.5 跨局熟人系统 📋 M3
每局从上一局幸存 NPC 中抽 1~2 个进入本局,带着记忆("玻璃桥上你推的是我兄弟")。
只需持久化一两个 NPC 的记忆切片,成本接近零,
但把单局社交决策变成跨局因果——背叛的长期代价从"本局复仇标记"
延伸为"下局开局就有人等着你"。活人感最便宜的来源。

## 4. 优先级结论

| 项 | 落点 |
|---|---|
| 守卫搜索态 / 开枪前摇 / 噪音圈 / 地标与出口指示 | ✅ M0 收尾(本次提交) |
| 广播插播规则变更 | M1,跟木头人一起验证 |
| 遗产时刻、跨局熟人 | 写进文档,M3 社交深水区实施 |
| 好感模糊化、脸特写降级 | M3 前的设计决定,不产生当期工作量 |
| 守卫制服线 | M5 后,精彩但不是骨架 |
