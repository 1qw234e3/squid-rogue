# 《孤岛 48》设计文档(细化稿 v0.2)

> 像素俯视角 Roguelike × 死亡游戏生存。一句话:48 人登岛,广播一响全员进入随机致命小游戏;局内靠搜刮、结盟、作弊活命,局外用奖金养成,再来一局。
> 骨架来源:元气骑士(手感、武器、局外养成)+ 死亡游戏题材(轮次结构)+ 轻量社交模拟(活人岛)。

## 0. 体验支柱与开发假设

四条支柱,所有后续设计争议以此裁决:

1. **呼吸式节奏**:松(沙盒密谋)与紧(游戏生死)交替。沙盒时长逐轮递减,窒息感递增。
2. **规则可被钻空**:每个关卡在"正面通关"之外,至少有一条情报、道具或社交的歪门邪道。
3. **人比关卡危险**:NPC 既是资源也是炸弹,社交决策的后果跨轮次回响。
4. **死了也有收获**:任何一局都推进局外进度或剧情线索;入岛到第一场游戏 ≤ 7 分钟,保证"再来一局"的冲动。

开发假设:1~2 人独立开发;PC 优先(手柄/键鼠),预留移动端虚拟摇杆;引擎建议 Godot 4(理由见 §10)。

## 1. 单局流程与节奏

### 1.1 时间轴(标准局,活到决赛约 38~42 分钟)

| 阶段 | 时长 | 内容 | 存活 |
|---|---|---|---|
| 入岛 | 1 min | 角色确认、广播开场、发 100 配给券 | 48 |
| 沙盒 R1 | 6 min | 自由活动 | 48 |
| 游戏 R1 | 3~5 min | 大逃杀型(木头人 / 抢椅子) | 48→30 |
| 沙盒 R2 | 5 min | | 30 |
| 游戏 R2 | 3~5 min | 个人技巧型(抠糖饼) | 30→18 |
| 沙盒 R3 | 4 min | | 18 |
| 游戏 R3 | 3~5 min | 信息博弈型(玻璃桥) | 18→10 |
| 沙盒 R4 | 3 min | | 10 |
| 游戏 R4 | 3~5 min | 社交清算型(弹珠 / 拔河) | 10→4 |
| 决赛 | 4~6 min | 竞技场 | 4→1 |
| 结算 | 1 min | 奖金、图鉴、线索入库 | — |

死亡处理:玩家死亡后二选一——立即结算退出,或"灵魂观战"到局末(能看到盟友的结局,补全 NPC 图鉴)。多数 run 实际只有 10~25 分钟,这才是 Roguelike 的真实节奏;30~40 分钟是"打满"的时长。

### 1.2 奖金池与个人账户

- **大池**:每淘汰 1 人 +100 奖金,总池 4700,冠军全取;逃岛结局放弃大池,换固定 2000 + 专属解锁。
- **个人账户**:沙盒成就、搜刮折算、关卡内表现即时入账(上限约 300/局),**死亡也能带走**——这是"死了也有收获"的经济保底。
- **广播系统**是节奏器兼叙事者:倒计时预告、规则宣读、淘汰播报("第 17 号参赛者,淘汰",全岛回响)、池子上涨音效,以及夹在电波杂音里的隐藏剧情线索。广播是本作的听觉签名。

## 2. 海岛沙盒

固定手工地图,约 120×80 格(16 px/格),小而密。地面层 7 个分区 + 通风管"夹层地图"(独立图层,8~10 个出入口;贴着房间的管段可触发偷听)。

### 2.1 分区

| 区域 | 功能 | 风险 | 典型机会 |
|---|---|---|---|
| 宿舍 | 床位储物、夜谈结盟 | 低 | 偷窃、密谈、私藏点 |
| 食堂 | 配给、斗殴、传闻集散地 | 中 | 食物(硬通货)、押注、刷好感 |
| 医务室 | 治疗、药剂 | 中 | 兴奋剂、镇静镖原料 |
| 广场 | 集合、公告、处刑展示 | 低 | 信息、威慑演出 |
| 仓库 | 大宗搜刮 | 中 | 工具道具、黑市入口候选 |
| 守卫区 | 情报、武器、伪装 | 高 | L3 情报、契约守卫、排班表 |
| 码头(常锁) | 逃岛终点 | 极高 | 剧情 |

### 2.2 行动动词集

对话 / 赠予 / 交易 / 结盟邀请 / 偷窃 / 跟踪 / 搜刮 / 潜行(蹲伏)/ 攻击 / 藏匿 / 使用道具。
掏出武器即进入敌对状态,守卫 5 秒内响应;沙盒杀人可行但代价极高(全员好感下降、被通缉),留给"灭口夺情报"的高风险流派。

### 2.3 机会事件(治"沙盒没事干")

每个沙盒阶段从事件池激活 3~5 个,以"传闻"形式进入情报面板,只给模糊提示("有人看见医生在仓库鬼鬼祟祟"),需实地确认。示例池:守卫换岗空档(守卫区大门 40 秒无人)/ 食堂斗殴(可押注或拉架)/ 黑市开张 / NPC 密谋(偷听到 X 与 Y 计划在玻璃桥推人)/ 补给空投 / 处刑公告(围观可白嫖情报)。

### 2.4 守卫与禁闭

守卫按排班巡逻(排班表可偷),视锥 120°、6 格。违规被抓:没收违禁品 + 禁闭(跳过剩余沙盒时间,被直接押进下一关、零准备)。三次累犯或袭警失败 = 当场处决。设计意图:惩罚是"失去机会"而非即死,鼓励玩家冒险。

### 2.5 情报系统(三级精度)

| 级别 | 内容示例 | 获取途径 | 关内效果 |
|---|---|---|---|
| L1 类别 | "下一关考反应" | 50 券 / 偷听 | 心理准备、预选道具 |
| L2 详情 | "木头人,双娃娃变体" | 150 券 / 好感≥50 的守卫 | 针对性备战 |
| L3 答案 | 扫描节奏 / 桥面前 4 块 / 最优描线 | 300 券 / 契约 / 盗窃 | 关卡内出现作弊 UI |

情报可转卖,也可造假——把假情报卖给敌对 NPC 是老千流的核心乐趣。

### 2.6 黑市

入口每局随机(通风管节点 / 后厨 / 仓库夹墙),进入需引荐(任一好感 ≥60 的 NPC 带路)或 200 券门票。每轮库存 5 件:违禁道具、L2/L3 情报、关卡神器(扫描镜等)、契约中介。

### 2.7 局内经济与偷窃

**配给券**是局内唯一流通货币(死亡清零)。来源:开局 100、搜刮、交易、斗殴押注、替 NPC 跑腿(顺带涨好感)。
偷窃 = timing bar 三段判定(完美 / 成功 / 被发现)。被发现:目标好感 −30、区域广播(周围 NPC −10)、偷守卫物资则直接通缉。

## 3. 死亡游戏关卡

**统一关卡接口**:每关 = 基础规则 + 变体(每局随机挂 1 个)+ 道具接口 + 社交接口 + 作弊线 + 失败演出。
**抽取约束**:每局抽 4 关 + 固定决赛;R1 必为大逃杀型、R4 必为社交清算型、同型不重复——体验骨架稳定,内容随机。

### 3.1 木头人(大逃杀型)

- 场景:纵向 60×24 格,起点线→终点线,散布可破坏掩体(木箱 HP3,被狙 3 次即碎)。巨型娃娃居顶。
- 节奏:绿灯 2.5~5 s(随机)→ 转头预警音 0.4 s → 红灯扫描 1.5~3 s。
- 判定:红灯期速度向量超阈值(摇杆死区 0.15)累计 0.2 s → 激光预瞄 0.5 s(此窗口内进掩体可免)→ 狙杀。角色带 0.3 s 惯性滑行——**预判松杆是本关手感核心**。
- NPC 表现:按勇气参数决定激进度;怂包可能赖着不动(总限时 120 s 逼迫全员);狠毒 NPC 会把人推出掩体当诱饵。
- 变体:① 双娃娃异步扫描 ② 夜战(手电视野)③ 传送带地块(红灯时必须借掩体)。
- 道具:消音鞋(红灯期可 0.5 格/s 蠕动)、烟雾弹(挡视线 2 s,全场受益)、肾上腺素(绿灯移速 +50%)。
- 情报 L3:界面显示扫描倒计时条。

### 3.2 抠糖饼(个人技巧型)

- 操作:右摇杆控制针尖沿轮廓描线;偏离轮廓 >2 px 开始累积裂纹值(蛛网裂纹实时可视化),裂满即碎 = 淘汰。速度↔稳定权衡:针尖抖动噪声幅度 ∝ 移动速度。
- 心理战:场上每有人碎掉,全员屏幕轻震 + 裂纹微增——连锁恐慌,人越死越手抖。
- 形状池:圆(容差高)/ 三角 / 星 / 伞(最难),随机分配;沙盒阶段可偷看自己的签,也可**换签栽赃**。
- 舔糖:长按暂停描线、缓慢回复裂纹,但总时限不停——典型风险决策。
- 变体:① 双层糖饼(描两圈)② 震动桌面(全程底噪)③ 信任接力(与盟友各描一半)。
- 道具:针头(降速增稳)、镇定剂(噪声 −40%,持续 10 s)。

### 3.3 玻璃桥(信息博弈型)

- 结构:2×16 块(变体 12~20),每对一真一假,由局种子决定。过桥次序由沙盒"抽号事件"决定,**号码本身是可交易道具**——后排号是硬通货。
- 操作:跳跃 = 朝向 + 0.3 s 蓄力确认(防误触)。已碎的块永久可见——记忆压力转化为信息公地。
- 推人:接近 + 互动键,1 s 前摇;被推者获得 0.4 s QTE 反抓,成功则挣脱并触发全岛广播(推人 = 重罪级声誉事件)。
- 节奏器:总限时 3 min,守卫从桥尾开始"清场射击"逼迫前进,治"桥头怂包堵路"。
- NPC:忠诚盟友(好感 ≥80)会主动替你探路;狠毒者伺机推人。
- 道具:扫描镜(透视 1 块,黑市稀有)、强化鞋(踩碎后 0.5 s 内可跳回)、记号笔(给后面的盟友留信息)。
- 变体:① 雾桥(只见前两排)② 整桥限时坍塌 ③ 三列桥。

### 3.4 弹珠(社交清算型)

- 配对:优先与好感最高者配对(= 你的盟友),无盟友则随机。**这是社交系统的账单时刻**。
- 玩法二选一(由 NPC 性格倾向决定,玩家可提议):
  - **A 物理弹射**:小场地回合制,双摇杆控制瞄准与力度,各 10 颗,弹对方出界或进洞,先输光者死。NPC 准度 = 技术参数。
  - **B 猜单双**:心理博弈。NPC 有性格驱动的"说谎模式"(投机者大注时 70% 说谎;忠诚者诚实但会给暗示),玩家通过对话选项 + **像素脸部特写帧**读破绽。
- 让局剧情:好感 ≥90 的忠诚 NPC 会主动输掉(经典弹珠悲剧,全作情感重锤)。
- 出千:磁石弹珠 / 灌铅骰,成功率 = 技术参数;失败 = 对方夺走你全部弹珠(直接濒死)。
- 结果广播:"X 出千害死了盟友"全岛回响 → 声誉系统介入(§4.3)。

### 3.5 拔河(团队清算型)

- 分组:按结盟阵营自动成队,落单者随机塞队——**沙盒结盟质量的硬考核**。
- 玩法:节奏 QTE(划船式),团队同步条;总拉力 = Σ(成员力量 × 节拍命中率),NPC 命中率 = 技术参数。玩家技能"喊口号":短时全队命中 +15%。
- 策略层:沙盒阶段可查看 NPC 力量值(提前招募壮汉)、给对方绳子抹油(对方全队命中 −10%)、收买对方成员放水。
- 失败队坠落全灭,但玩家保留一次超难"抓住边缘"QTE 作为最后一搏(防纯团灭挫败)。

### 3.6 抢椅子(原创,大逃杀型)

- 圆形竞技场,中央椅子数 = 存活数 − 本轮淘汰额。音乐期全员必须保持移动(站定 >1.5 s 吃狙击警告,不许蹲点);音乐在随机 8~20 s 后停 → 椅子激活,站上 0.5 s 完成锁定。
- 清场期:未坐者面对从场边刷出的元气骑士式弹幕(环形波 + 追踪弹),期间随机再升起 1~2 把"补刀椅"——绝境逆转点。
- 对抗:近战推撞可把人撞离椅子(锁定进度被打断)。道具:推撞手套、椅子嗅探器。
- 变体:① 移动椅 ② 假椅(坐上即爆)③ 音乐由某个 NPC 控制(**贿赂线**)。

### 3.7 守卫猎杀(原创,战斗型)

- 全游戏唯一程序化地图(BSP 迷宫),与固定海岛图形成对照。玩家与幸存 NPC 撒入,出口 1~3 个,限时 4 min。
- 守卫小队 AI:巡逻 → 发现(视锥 + 声响)→ 追捕 → 呼叫包抄。开局徒手,武器从迷宫箱子里捡——**双摇杆战斗全开,这一关就是完整的元气骑士**。
- 反杀守卫掉落:捷径钥匙、伪装面具(15 s 免警觉)、武器。
- 社交接口:盟友会开门等你;敌对 NPC 会故意把守卫引向你。

### 3.8 决赛:竞技场

- 4 人混战,全武器开放,毒圈 = 守卫狙击红线收缩,场内散布武器与补给。
- **分赃提案**:若决赛圈存在好感 ≥90 的盟友,触发"平分"事件——双方各自按下确认才生效(奖金减半、双双存活)。投机型 NPC 可能在你按下的瞬间偷袭——全作最戏剧化的一秒。
- 若已集齐逃岛线索,决赛可整体替换为逃脱序列(§8)。

## 4. NPC 与社交系统(工作量大头)

### 4.1 原型与参数

8 种性格原型 × 五维参数(勇气 / 贪婪 / 忠诚 / 技术 / 魅力,各 1~10)+ 随机外观与编号 → 每局从 100+ 人物池抽 47 个同场。

| 原型 | 行为倾向 | 对玩家的意义 |
|---|---|---|
| 忠犬 | 结盟后绝不背叛,会让局、挡刀 | 最优长期投资 |
| 投机商 | 唯利是图,情报贩子,大注必算计 | 好交易对象,别带进弹珠局 |
| 毒蛇 | 主动害人、推人、栽赃 | 威胁源,但也可雇佣 |
| 怂包 | 龟缩,桥头堵路,易被勒索 | 廉价票仓 / 人肉盾 |
| 疯子 | 混沌行为,搅乱所有局 | 不可控变量,制造名场面 |
| 老兵 | 技术高、冷漠、独行 | 难结盟,决赛大敌 |
| 圣母 | 救人、分食物 | 易刷好感,关键时刻心软 |
| 卧底守卫 | 隐藏身份 | 黑幕剧情线入口 |

### 4.2 好感度(0~100)与记忆

| 区间 | 状态 | 效果 |
|---|---|---|
| 0~19 | 敌对 | 主动针对、引守卫、推人优先选你 |
| 20~49 | 中立 | 正常交易 |
| 50~79 | 友好 | 帮小忙、交易折扣、共享传闻 |
| 80~100 | 生死之交 | 团队关掩护、弹珠让局、决赛分赃 |

好感事件表(节选):赠送食物 +5 / 共享情报 +10 / 团队关救命 +25 / 完成跑腿 +8 / 偷窃败露 −30 / 栽赃败露 −40 / 背叛 −100(锁死敌对并打上复仇标记)。
每个 NPC 持有 ≤10 条事件记忆,对话中会引用("食堂那次多亏你")——**这是"活人感"的关键来源**。

### 4.3 NPC↔NPC 模拟与声誉广播

- 沙盒阶段每 30 s 一次抽样 tick:随机 NPC 两两发生结盟 / 交恶事件,系统维护一张阵营图(并查集 + 边权)。玩家可旁观、挑拨、加入既有阵营。
- 声誉广播:推人、出千、背叛等重罪事件全岛广播,所有 NPC 按各自性格修正对你的好感(毒蛇反而 +,圣母大幅 −)。
- 复仇标记:被你背叛且存活的 NPC,在后续所有关卡中优先针对你,直到一方死亡——背叛的长期代价具象化。

### 4.4 对话生产方案(防内容黑洞)

模板组合制:8 性格 × ~30 事件类型 × 3 好感层 ≈ 先写 **200 条核心模板** + 变量填充,设硬性上限。LLM 实时生成(本地小模型或 API)留作可选增强——成本、延迟、可控性三关都验证过再上,MVP 阶段不依赖。

### 4.5 性能方案:三层 LOD

- 近景(同屏 ≤12 个):完整行为树,5 Hz 决策频率。
- 远景(同岛屏外):简化状态机 + 每 30 s 掷一次事件骰。
- 抽象(游戏关内不同场者):直接按参数掷存活结果,只播报结局——**48 人不必真的跑 48 份抠糖饼模拟**。

## 5. Roguelike 系统

### 5.1 随机性清单

关卡序列与变体 / 规则参数(桥长、扫描节奏)/ NPC 阵容与初始阵营 / 道具与黑市库存 / 机会事件 / 隐藏事件 / 黑市位置 / 通风口开闭。全部由单一 RunSeed 派生(支持种子分享与挑战)。

### 5.2 局内构筑:道具三类

**消耗品(首发 20+)**:烟雾弹、肾上腺素、针头、镇静镖(命中的 NPC 下一关参数 −2)、伪造引荐信、假情报、磁石弹珠、消音鞋(3 次)等。

**被动宝物(首发 30+,优先做"改规则"的)**:
- 千里眼:所有情报自动 +1 级
- 赌怪:猜单双时显示对方"心跳像素条"
- 幽灵步:红灯判定宽限 +0.3 s
- 团宠:同队 NPC 节拍命中 +10%
- 秃鹫:同场每死 1 人,+5 配给券
- 老六:蹲伏时对 NPC 完全隐身(守卫除外)
- 替罪羊:每局首次致死判定转嫁给同场好感最低的 NPC(作恶成就解锁)

**武器(首发 15+)**:仅在守卫猎杀、决赛、逃岛序列、敌对状态下可用;弩为无声武器,是沙盒暗杀流专属。

### 5.3 守卫契约(风险换确定性,Roguelike 经典"恶魔交易")

1. 保送:下一关免死 1 次,代价决赛大池 −30%。
2. 黑吃黑:指定没收某 NPC 一件道具归你,但对方必然知道是你干的。
3. 睁只眼:本轮沙盒违规免责,代价"欠人情"——之后随机时刻守卫提要求,拒绝即通缉。

## 6. 操作与战斗(双摇杆)

左摇杆 8 向移动(4 格/s),右摇杆瞄准 + 扣发,翻滚 0.4 s 无敌帧(CD 1.5 s),交互键,道具轮盘。
手感基准对标元气骑士:开火 2 px 屏震 + 抛壳,命中 0.05 s hitstop,枪口火光 2 帧。**M0 里程碑的唯一目标就是把这套手感做出来,做不出来不往下走。**
沙盒内武器默认收起,掏枪 = 敌对宣告。

## 7. 元进度(局外养成)

货币:**奖金**为唯一局外货币 = 冠军大池 / 个人账户 / 成就奖励。

角色(首发 8 个,示例):

| 角色 | 主动技能 | 被动 | 解锁条件 |
|---|---|---|---|
| 失业者(初始) | 重掷:任一随机结果 reroll(1 次/局) | 赌博类游戏提示+ | — |
| 医生 | 急救:救活自己或 NPC 一次 | 医务道具半价 | 2000 奖金 |
| 前特种兵 | 短冲刺 | 守卫视野对其 −20% | 3000 奖金 |
| 小偷 | 必成功偷窃(1 次/轮) | 通风管移速+ | 成就:累计偷窃 30 次 |
| 老千 | 换签(糖饼形状 / 过桥号码) | 出千成功率+ | 成就:出千赢 3 局弹珠 |
| 主播 | 嘲讽:吸引全场仇恨 3 s | 个人账户收入 ×1.5 | 4000 奖金 |
| 老兵 | 武器精通 | 决赛开局自带武器 | 5000 奖金 |
| 逃亡者 | 撬锁 | 线索掉率 ×2 | 达成逃岛结局 |

设施(局外大厅):
- **训练场**:单独练习任意已见过的小游戏——挫败感的出口,必做。
- 情报屋:每局开局赠送 1 条 L1 情报。
- 军械库:扩充守卫猎杀 / 决赛的武器池。
- **档案馆**:NPC 图鉴,记录每个见过的 NPC 及其死法(情感钩子)+ 逃岛线索墙。
- 宿舍升级:开局道具栏 +1。

## 8. 隐藏结局:逃出海岛

跨局收集 4 条线索(档案馆持久保存):海图碎片(守卫区保险柜)/ 巡逻排班表(扒军官身)/ 断电方案(电工 NPC 好感 ≥80 相赠)/ 船库钥匙(特定 NPC 的弹珠局遗物)。
集齐后,任意局自 R3 起沙盒阶段出现"今夜行动"选项 → 15 min 逃脱序列:断电 → 通风管潜行 → 码头遭遇战 → 守卫长 BOSS → 开船。
可携带好感 ≥80 的盟友(每多带 1 人难度上升,结局 CG 分支不同)。
结局矩阵:**冠军**(独吞大池)/ **逃脱**(固定 2000 + 专属角色)/ **黑幕**(集齐另一组档案线索、揭穿主办方真相——留作后续大版本)。

## 9. 美术、音频与 IP 边界

- 像素规格:角色 16×24,逻辑分辨率 640×360,整数倍缩放。
- 色彩策略:岛内压灰蓝低饱和,游戏场高饱和撞色——"游乐园式恐怖"的反差感。
- **IP 注意**:木头人、弹珠、拔河、抢椅子都是传统公共游戏,玩法层没有问题;但"圆三角方面具 + 粉色连体服 + 编号运动服"这套视觉组合是 Netflix 的强识别资产,上架前务必做原创化。建议的原创视觉锚点:**深海 / 灯塔主题**——守卫 = 老式潜水铜盔,编号 = 船员铭牌,广播 = 灯塔雾笛。既规避风险,也能和你海角那套深海蓝审美一脉相承,顺手建立个人视觉签名。
- 音频签名:广播女声宣读规则、雾笛切换轮次、濒死心跳、糖饼"咔"的裂响。

## 10. 技术实现(Godot 4 建议方案)

- 选型理由:2D 一等公民;GDScript 语法接近 Python(你的主力语言);场景系统天然适配"分区地图 + 独立小游戏关";一键导出 Win/macOS,后续可出移动端。备选:Unity(资产生态)、Phaser/TS(网页发行快,但 48 NPC + 弹幕的性能要谨慎)。
- 架构:
  - `RunManager` 状态机:Lobby → Sandbox(n) → Briefing → MiniGame(n) → Settlement。
  - `EventBus`(autoload 单例):淘汰、好感变化、传闻、广播事件,全系统解耦。
  - NPC = `NPCData`(Resource:原型 / 参数 / 记忆)+ `NPCAgent`(行为树)。
  - 小游戏统一接口:`setup(seed, variant, participants)` + `finished(survivors)` 信号;每关独立场景,可单独运行测试、并行开发。
- 性能:NPC 三层 LOD(§4.5);弹幕用 MultiMesh + 对象池,单屏 ≤400;沙盒寻路用 NavigationRegion 预烘焙。
- 确定性:RunSeed 派生所有子系统种子,支持复盘和种子分享。
- 存档:局外进度 JSON;局内只做暂停快照(序列化 RunState)。

## 11. 路线图与风险

| 里程碑 | 周期 | 内容 | 验收问题 |
|---|---|---|---|
| M0 手感切片 | 2 周 | 双摇杆射击 + 守卫追捕迷宫 | 打枪爽不爽? |
| M1 垂直切片 | 4~6 周 | 8 NPC、食堂+宿舍、木头人+守卫猎杀、完整一局 15 min | 循环成不成立? |
| M2 内容扩张 | 4 周 | 糖饼 / 玻璃桥、情报系统、结盟 v1 | 沙盒有没有事干? |
| M3 社交深水区 | 6 周 | 弹珠、背叛/广播/复仇、黑市、48 人 LOD | NPC 像不像活人? |
| M4 完整循环 | 6 周 | 抢椅子 / 拔河 / 决赛、元进度、8 角色 | 可对外测试 |
| M5 打磨 | — | 逃岛结局、变体填充、数值平衡、商店页 | EA 发布 |

风险清单(按杀伤力排序):

1. **对话量黑洞**:社交系统的文本量会无限膨胀,模板组合硬上限 200 条起步,LLM 后置。
2. **沙盒空转**:机会事件每轮保底 3 个 + 倒计时压迫;M2 必须灰盒测试,沙盒不好玩则整个松紧节奏崩塌。
3. **表演性优先于模拟性**:玩家只需要"觉得 NPC 活着"——可见的小动作(递东西、靠墙、追逐)性价比远高于真模拟。
4. **关卡量产陷阱**:8 个关卡 = 8 个独立小游戏项目;先用 2 关(木头人 + 守卫猎杀)验证"俯视角翻译公式",跑通再量产。
